# CHORE-02: Remove tracked sandbox/node_modules/

## Problem / Goal
The `sandbox/node_modules/` directory is currently tracked in Git, which bloats the repository and causes unnecessary diffs. It should be removed from Git and ignored.

## Current Behavior
`git ls-tree` reveals hundreds of files under `sandbox/node_modules/`.

## Expected Behavior
`sandbox/node_modules/` should be absent from the tracked file list. Local installation should still be possible via `.gitignore`.

## Proposed Test Case (TDD)
- [x] CI/Unit Test: `git ls-files sandbox/node_modules/` should return no results.

## Context / Constraints
- Ensure `.gitignore` is updated if necessary.
- Use `git rm -r --cached sandbox/node_modules/`.
