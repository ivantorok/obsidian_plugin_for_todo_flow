# Discovery: Unverified Content

This directory is the designated staging area for **unverified content**. It is intentionally isolated from core processes (backlog, user stories, and protocol) to ensure raw ideas are thoroughly vetted before they influence the system.

## Workflow
1. **Raw Intake**: Drop notes, snippets, or external feedback here.
2. **Governor's Scan**: The Process Governor (PG) periodically scans this folder for "orphans".
3. **Verification**: Content must be verified and refined here.
4. **Promotion**: Once verified, the PG triages items into the official `backlog/` or updates the `[[CONCEPT_ATLAS]]`.

## Constraints
- Do **not** link core documentation to files in this directory.
- This is a "low-stakes" environment for rapid capture.
