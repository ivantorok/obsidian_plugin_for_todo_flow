# Stage: Stack

The primary execution workbench where tasks are ordered, timed, and completed.

## Characteristics
- **Projection Principle**: The UI is a reactive projection of the Markdown file.
- **Sovereignty**: Owns the majority of global keybindings.
- **Mechanics Applied**:
    - `[[VIEWPORT_SOVEREIGNTY]]`: Central for mobile title editing and duration scaling.
    - `[[FOCUS_HANDSHAKE]]`: Ensures "The Stack is Home" (Greedy Focus).

## Components
- **Full**: `StackView.svelte` (Desk).
- **Lean**: `LeanStackView.svelte` (Mobile).
    - Features `[[CIRCULAR_LOOP]]` and `[[VICTORY_LAP]]`.
