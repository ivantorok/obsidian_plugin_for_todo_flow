# Environment: Power Desktop

Defines the behavior for the full-featured desktop experience.

## Key Features
- **Multi-View Orchestration**: Simultaneous display of Dump, Triage, and Stack.
- **Home Row First**: Keyboard-dense interaction model.
- **Fluid Gestures**: Drag-and-drop reordering and context menus are first-class citizens.

## Mechanics Applied
- `[[FOCUS_HANDSHAKE]]`: Critical for managing keystrokes across multiple visible views.
- `[[READ_MERGE_WRITE]]`: Handled with more explicit user prompts (Modals).
